What Is a Ratio?

A ratio is a mathematical comparison between two numbers. It shows how many times one value contains another. Ratios are usually written in the format A : B.

When two ratios are equal, they form a proportion, written as: A : B = C : D. Using a proportion calculator, you can quickly determine the missing value in this equation.
