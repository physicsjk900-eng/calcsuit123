What Is a Square Root?

The square root of a number is a value that, when multiplied by itself, produces the original number. For example:

  • √4 = 2 (2 × 2 = 4)
  • √9 = 3 (3 × 3 = 9)
  • √16 = 4 (4 × 4 = 16)

The principal square root refers to the non-negative square root of a number. Using an online square root calculator, you can easily find the square root of any real number without doing manual calculations.
